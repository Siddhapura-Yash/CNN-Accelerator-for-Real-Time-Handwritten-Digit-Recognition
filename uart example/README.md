## Description

This example demonstrates UART-based image transfer for the CNN accelerator. A 28×28 grayscale image is transmitted from a PC to the FPGA at **1 Mbps (10 lakh baud rate)**, processed by the CNN, and the predicted digit is displayed on a seven-segment display.

The design was tested on an **Efinix Ti375 FPGA** running at **50 MHz**.
